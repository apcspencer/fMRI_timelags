# fMRI_timelags

Code for cross-correlation between fMRI data and a timeseries.

This code accompanies the following manuscript:
Inès de Riedmatten, Arthur P C Spencer, Jasmine Nguyen-Duc, Jean-Baptiste Pérot, Filip Szczepankiewicz, Oscar Esteban, Ileana O Jelescu; **Evaluating the dependence of ADC-fMRI on haemodynamics in breath-hold and resting-state conditions**. Imaging Neuroscience 2025; doi: https://doi.org/10.1162/IMAG.a.1020

If you use this code, please cite the manuscript above.
